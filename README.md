## Installation

- Clone This Front End Repo

```
git clone https://github.com/Chandra-Kurnia/React-Tuku.com.git

```

- Go To Folder Repo

```
cd React-Tuku.com
```

- Install Module

```
npm install
```

- Type ` npm run dev` To Start Website
